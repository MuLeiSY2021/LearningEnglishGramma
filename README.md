# LearningEnglishGramma

巴别塔式英语语法学习系统 — 一座从地基到塔顶的五层语法之塔。

## 这是什么？

一个用 [Obsidian](https://obsidian.md) 构建的英语语法知识库，专为**中文母语者**设计，面向**考研英语**长难句阅读能力。

核心理念：
- **发现式学习** — 不是死记规则，而是通过谜题自己发现规律
- **中文桥梁** — 每个语法点都从中英差异出发，利用母语直觉
- **五层递进** — 从句子骨架到高阶修辞，层层拆解

## 五层塔结构

```
┌─────────────────────────────────────┐
│  第五层 · 高阶结构                    │  倒装/分裂句/名词化/花园路径句
├─────────────────────────────────────┤
│  第四层 · 句间关系                    │  信息结构/衔接/并列与从属
├─────────────────────────────────────┤
│  第三层 · 从句世界                    │  关系从句/名词从句/状语从句/虚拟语气
├─────────────────────────────────────┤
│  第二层 · 修饰与扩展                  │  名词短语/介词短语/非谓语/时态
├─────────────────────────────────────┤
│  第一层 · 句子骨架                    │  五大基本句型 (SV/SVO/SVC/SVOO/SVOC)
└─────────────────────────────────────┘
```

## 数据量

- **26** 篇概念笔记（全部完成）
- **21** 个谜题（含渐进提示和详细解答）
- **3** 篇中英对比桥梁笔记
- **2** 种特殊题型（错误辨析 / 跨层综合段落全析）

## 如何使用

1. 用 Obsidian 打开本文件夹作为 Vault
2. 从 `00-系统/学习路径.md` 开始
3. 按层级递进，每层先读概念笔记，再做谜题
4. 在 `60-发现笔记本/` 记录你的语法发现

## 设计原则

- 每篇概念笔记遵循统一模板：直觉入口 → 中文桥梁 → 规则揭示 → 实例分析 → 关联 → 自检
- 谜题提供三级提示（轻/中/重），鼓励先独立思考
- 所有笔记通过 `[[双链]]` 互相关联，支持非线性探索
- 语料全部来自真实文本（考研阅读、The Economist、Nature、文学作品）
